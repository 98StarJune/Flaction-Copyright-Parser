# Flation Copyright Parser

Flation (www.flaticon.com)에서 저작권 정보를 Parsing 합니다.

### 입력
1. /TextFile 디렉토리에서 input.xml을 엽니다.
2. '<a href="...", title="..."> ... </a>' 형태의 xml을 입력합니다.

### 실행
- ```npm install```
- ``` npm run compile```
- ```npm run start```
- 위 명령어를 순서대로 수행합니다.

### 결과
/TextFile 데릭토리에서 output.txt에 '이름 (저작자명)'이 한줄로 정리됩니다.
